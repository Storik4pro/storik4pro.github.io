Быстрый старт

Установка обычной версии

  1. Скачайте последнюю версию goodbyeDPI UI используя страницу релизов.
  2. Выключите антивирус.
  3. Установите goodbyeDPI UI.
  4. Добавьте файл goodbyeDPI.exe в список исключений вашего антивируса.
  5. Включите антивирус.
  6. Вы великолепны!

Установка portable версии

  1. Скачайте архив, используя страницу релизов.
  2. Разархивируйте архив в удобное вам расположение.
  3. Добавьте goodbyeDPI.exe в список исключений вашего антивируса.
  4. Запустите goodbyeDPI.exe.
  5. Установка завершена!

Обновление до версии 1.0.4 с помощью Update Assistant

  1. Скачайте update.exe, используя страницу релиза 1.0.4.Так же можете использовать прямую ссылку
  2. Поместите файл update.exe в папку с goodbyeDPI.exe.
  3. Запустите файл.
  4. Следуйте инструкциям установщика.
  5. Обновление завершено!
⚠

Обратите внимание: Если обновление через Update assistant вызывает ошибки, то следует обратиться к FAQ Update assistant Перейти на страницу поддержки Update assistant
