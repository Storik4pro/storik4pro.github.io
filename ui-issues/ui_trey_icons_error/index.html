Ошибка: множество иконок в трее

Данная ошибка может произойти после запуска программы через галочку "запустить после завершения" в установщике. Для исправления этого поведения следует завршить процесс программы goodbyeDPI.exe в диспетчере задач и запустить программу заного

⚠

Обратите внимание: в версиях 1.0.2 и выше эта ошибка была исправлена
