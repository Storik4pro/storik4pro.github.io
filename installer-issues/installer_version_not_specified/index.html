Уведомление при запуске Update assistant

Обычно такое уведомление появляется, если вы решили запустить Update assistant вручную. В таком случае, установка обновления будет выполнена в рабочую директорию update.exe. Версия для установки будет выбрана последняя доступная на странице релизов.

⚠

Обратите внимание: при запуске Update assistant вручную сохранение данных goodbyeDPI-UI не осуществляется. После завершения обновления следует вручную изменить неообходимые праметры в settings.ini. Подробности см. в FAQ goodbyeDPI-UI

Если вы хотите выбрать определенную версию и/или директорию, то вам следует запустить update.exe из PowerShell с определенными параметрами запуска.

.\update.exe -l {language} -v {version} -d {directory}

Где {language} - язык приложения EN или RU; {version} - версия для установки; {directory} - директория для установки обновления

Например, код приведенный ниже запускает обновление до версии 1.0.3 в директории "C:\Program Files\GoodbyeDPI_UI", с языком интерфейса "русский"

.\update.exe -l RU -v 1.0.3 -d "C:\Program Files\GoodbyeDPI_UI"
⚠

Обратите внимание: запуск обновления в директориях, закрытых для изменения (например, в папке Program Files) потребует запуска Update assistant с привелегиями администратора.
